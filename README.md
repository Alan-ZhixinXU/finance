# finance

#### 项目介绍
一个简单的财务会计软件，能够录入会计凭证，并根据凭证自动生成三大报表。详情查看：https://www.cnblogs.com/edwardorchis/p/10506391.html

#### 软件架构
WPF的客户端+自宿主.Net Web API
